# Almanac Sarawak 2026

An all-in-one Android application designed for Sarawakians, featuring a comprehensive 2026 calendar with local public holidays, school holidays, and astronomical data. The app also includes productivity tools and resources for learning Generative AI.

## Technical Details

*   **Language:** Kotlin
*   **UI Framework:** Jetpack Compose (Material 3)
*   **Minimum SDK:** 26 (Android 8.0)
*   **Target SDK:** 36 (Android 15)
*   **Architecture:** MVVM-inspired (State hoisting with Compose)
*   **Local Storage:** SharedPreferences (via Gson for object serialization)
